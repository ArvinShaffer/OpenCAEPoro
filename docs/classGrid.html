Detailed Description

Grid contains basic information of grids of reservoir, the rock properties in grids are also included. all grids are stored here, you can regard it as a database of reservoir. considering the existence of inactive grids(whose volume of pores is too small or effective resource is too little) or switch of activity of grid, Grid class is necessary. Grid class is static while simulating, active grids will be stored in bulks, which is "area" for calculating.
