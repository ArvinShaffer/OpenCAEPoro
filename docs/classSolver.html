Detailed Description

template<typename T>
class Solver< T >

A template class designed to stores and solves linear system derived from discrete method. the maxtrix is stored in the form of row-segmented CSRX internaly, whose sparsity pattern is almost the same as neighbor in Connection_BB. if martix is block matrix, then the T will be a vector.
